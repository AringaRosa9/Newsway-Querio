# 用户指南 / User Guide

## 功能概览

AI News Search 是一个智能新闻搜索引擎，提供以下核心能力：

1. **智能搜索** — 混合检索（关键词 + 语义），返回相关新闻和 AI 生成摘要
2. **对话搜索** — 通过多轮对话探索新闻，支持追问和上下文理解
3. **事件追踪** — 自动聚合相关新闻为事件，展示时间线
4. **订阅推送** — 订阅关键词/话题，获取新闻推送通知
5. **跨语言搜索** — 中英文互搜，自动翻译查询
6. **行业搜索** — 金融/科技专业搜索模式

---

## 搜索

### 基础搜索

在搜索框输入关键词，按 Enter 或点击搜索按钮。

示例查询：
- `AI 最新进展`
- `Tesla stock price today`
- `半导体芯片 2025`

### 筛选

左侧面板支持按以下维度筛选：
- **时间范围**：今天 / 本周 / 本月 / 自定义日期
- **分类**：科技 / 财经 / 国际 / 社会 等
- **情感**：正面 / 中性 / 负面
- **来源**：指定新闻源
- **语言**：中文 / 英文

### AI 摘要

搜索结果顶部显示 AI 生成的综合摘要，包含引用来源标注 `[1]`、`[2]` 等，点击可跳转原文。

---

## 对话搜索

点击导航栏「对话」进入对话模式。

### 使用方式

1. 直接输入问题，AI 会自动搜索并综合回答
2. 支持追问："告诉我更多"、"这个事件后续如何？"
3. 对话历史自动保存，可在左侧面板切换

### 示例对话

> **你**：最近 AI 领域有什么重大进展？  
> **AI**：根据最新新闻报道... [搜索结果]  
> **你**：OpenAI 的具体情况呢？  
> **AI**：关于 OpenAI 的最新动态... [基于上下文的追问搜索]

---

## 事件追踪

点击导航栏「事件追踪」查看自动聚合的新闻事件。

- 支持切换时间窗口：24小时 / 72小时 / 一周
- 每个事件展示相关文章数量、情感分布、关键实体
- 时间线视图展示事件发展脉络

---

## 订阅

注册登录后，可以在「订阅」页面：

1. **创建订阅** — 设置关键词、类型（关键词/话题/事件）和频率（实时/每日/每周）
2. **管理订阅** — 开启/关闭、删除订阅
3. **查看通知** — 切换到「通知」标签查看新闻推送

---

## 行业模式

系统自动识别金融/科技类查询并切换专业搜索模式：

- **金融模式**：优先显示权威财经来源，识别股票代码/经济指标
- **科技模式**：优先显示科技媒体，识别版本号/技术术语

---

## 多语言

- 中文查询自动搜索中英文新闻
- 英文查询同样返回相关中文新闻
- 搜索结果标注语言来源

---

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Enter` | 发送搜索/消息 |
| `Esc` | 清空搜索框 |
| `Shift + Enter` | 对话模式中换行 |
